Instructions

First select the sources/projects, on which the rules shall be execute. Select parts of a project or a whole project. Right click to open Context menu. Under jSparrow menu select option Select rules to apply.

The second step is to decide which improvements shall be executed

  • String manipulation
  • Formatting
  • Coding conventions
  • Performance
  • Readability
  • Old language constructs
  • Lambda
  • Logging
    • Add desired rules and click Finish button to start refactoring.

    After applying the selected rules, jSparrow displays a diff-view with the old and the imporved sources. You still have the possibility to unselect rules and run jSparrow again. Only approved changes will be effected on your sources.
